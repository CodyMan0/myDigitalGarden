- presentational component
	- 사용자가 직접 보고 조작하는 컴포넌트 
	- state를 직접 조작하지 않고 container component가 내려준 Prop의 함수에 의해 State를 변경한다. 
- container component
	- 어떻게 동작하는지 어떤 로직을 수행하는지에 관련
	- markup을 사용하지 않고 스타일 사용하지 않는다.
