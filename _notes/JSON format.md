# XML -> JSON
이유 : 가독성 

```json
{
  "name": "홍길동",
  "age": 25,
  "married": false,
  "family": { "father": "홍판서", "mother": "춘섬" },
  "hobbies": ["독서", "도술"],
  "jobs": null
}
```


# JSON 내장 객체
브라우저 혹은 node.js 런타임에서 실행하든 전역에서 접근이 가능하다. 

1. JSON.parse() : 문자열을 JS 객체로 변환 
2. JSON.stringify() : JS 객체를 문자열로 변환
